Marcado de MediaWiki

Marcado MediaWiki

Para dar formato a una página de MediaWiki se pueden utilizar, entre otros, los siguientes códigos de marcado:

Texto
Resultado Sintaxis
cursiva ''ejemplo''
negrita '''ejemplo'''
negrita cursiva '''''ejemplo''''
subrayado <u>ejemplo</u>
tachado <s>ejemplo</s>
Sangría :ejemplo
Más sangría ::ejemplo :::ejemplo etc.
Superíndice <sup>ejemplo</sup>
Subíndice <sub>ejemplo</sub>
Ampliar <big>ejemplo</big>
Reducir <small>ejemplo</small>
Nuevo párrafo dejar un línea en blanco
Título de primer nivel = ejemplo =
Título de segundo nivel == ejemplo ==
Título de tercer nivel === ejemplo ===
Título de cuarto nivel ==== ejemplo ====
Título de quinto nivel ===== ejemplo =====
Título de sexto nivel ====== ejemplo ======
Líneas horizontales ----
texto que no debe interpretarse como formato de mediawiki <nowiki>ejemplo</nowiki>

Listas
Resultado Sintaxis
Listas no numeradas

* uno
* dos
Sublistas no numeradas * uno
** aa1
** aa2
* dos
Listas numeradas

# uno
# dos
Sublistas numeradas # uno
## aa1
## aa2
# dos

Enlaces
Resultado Sintaxis
Enlace interno [[ejemplo]]
Para que el texto del enlace interno sea distinto del nombre de la página [[ejemplo|texto visible]]
Enlace a página de desambigüación [[ejemplo (particular)|]]
Enlace a sección en la misma página [[#Sección]]
Enlace a sección en otra página [[Artículo#Sección]]
Enlaces externos http://www.example.org
Para que el texto del enlace externo sea distinto del nombre de la página [http://www.example.org ejemplo]
Redirección #REDIRECCIÓN [[destino]]
#REDIRECT [[destino]]
Enlace a categoría [[:Categoría:ejemplo]]
Incluir una página en una categoría [[Categoría:ejemplo]]

Firmas
Resultado Sintaxis
Nombre del usuario ~~~
Nombre del usuario y fecha y hora ~~~~
Fecha y hora ~~~~~

Imágenes
Resultado Sintaxis
Insertar imagen subida previamente [[Archivo:nombre.extensión|opciones|leyenda]]
[[Imagen:nombre.extensión|opciones|leyenda]]
[[File:nombre.extensión|opciones|leyenda]]
border y/o frameless, frame, thumb o thumbnail opciones de formato
anchura {width}px
altura x{height}px
anchura y altura {width}x{height}px
alineación horizontal left, right, center, none
alineación vertical baseline, sub, super, top, text-top, middle, bottom, text-bottom
enlace link={destino}

Tablas
Resultado Sintaxis
Comienzo de la tabla {|
Tabla con estilo mediawiki (fondo gris, borde fino) {| class="wikitable"
Tabla con controles para ordenar los elementos { class="sortable"
Leyenda de tabla (opcional) |+
Fila (opcional en la primera fila) |-
Celda de cabecera (<th>) (nota: se pueden escribir varias celdas en la misma línea separándolas por !!) !
Celda (<td>) (nota: se pueden escribir varias celdas en la misma línea separándolas por ||) |
Fin de la tabla |}
Se pueden añadir atributos a las celdas, filas o tabla separándolos por un |

Los atributos son los de las tablas de html (atributos comunes y atributos específicos)

 | align="right" | texto
