Aplicación: Joomla - Joomla 2.5.4 (02/04/12)

Nota: las capturas deben ser capturas de la pantalla completa en la que se vea la ventana del navegador y algún icono del escritorio.

1. Instalación

  • Descomprimir Joomla en una carpeta joomla accesible en el servidor local.
  • Entrar en phpMyAdmin como usuario root y crear un usuario de MySQL con nombre iwb_joomla, contraseña iwb_joomla y base de datos iwb_joomla.
  • Abrir joomla/ en el navegador. Se inicia el proceso de instalación:
    • Paso 1: Idioma

      Seleccionar el idioma: Spanish (español)

    • Paso 2: Comprobaciones

      Comprobar que los requisitos se cumplen

    • Paso 3: Licencia
    • Paso 4: Base de datos

      Tipo de base de datos: Mysqli

    • Paso 5: Configuración del FTP (no es necesario configurarlo)
    • Paso 6: Configuración principal

      Nombre del sitio: Examen final de suNombre susApellidos

      Su correo: admin@example.org

      Nombre del administrador: admin

      Contraseña del administrador: admin

      Datos de ejemplo: Datos de ejemplo predeterminados en inglés (GB) No instalar los datos de ejemplo

    • Paso 7: Finalizar

      Eliminar carpeta de instalación

  • Hacer dos capturas de pantalla:
    • una captura donde se vea la pantalla de inicio general joomla_1-1.png:

    • una captura donde se vea el formulario de entrada de administrador y guardarla como joomla_1-2.png:

2. Traducir Joomla

  • Traducir el panel de control a español
    • Entrar en Joomla con usuario admin
    • Entrar en Language Manager y comprobar que sólo está instalado el idioma inglés. Hacer una captura de pantalla del Language Manager y guardarla como joomla_2-1.png:

    • Copiar el archivos idioma_admin_2.5.4_es-ES.zip en la carpeta joomla/tmp y descomprimirlo en esa carpeta
    • Entrar en Extension Manager e instalar el paquete de traducción al español del panel de administrador, haciendo clic en el botón Examinar del apartado Upload Package File y seleccionando el archivo es-ES_admin_2.5.4.zip.
    • Volver al Language Manager y en la sección Installed - Administrator comprobar que se puede elegir el idioma español. Hacer una captura de pantalla del Language Manager y guardarla como joomla_2-2.png:

    • Seleccionar el idioma español y hacer una captura de la pantalla principal del panel de control y guardarla como joomla_2-3.png:

  • Traducir el sitio al español
    • Realizar la misma tarea pero para traducir los sitios, utilizando el archivo idioma_front_2.5.4_es-ES.zip.
    • Seleccionar el idioma español para los sitios y hacer una captura de la pantalla principal del sitio y guardarla como joomla_2-4.png:

3. Crear un usuario

  • Con el Gestor de usuarios, crear el usuario de nombre Usuario 1 y nombre de acceso y contraseña usuario_1. y asignarle en los grupos Author, Editor y Publisher

    Nota: No preocuparse por el aviso "No se puede inicializar la función de correo".

  • Entrar en el sitio como usuario_1 y hacer una captura de pantalla y guardarla como joomla_3-1.png:

4. Crear una copia de seguridad

  • Entrar en phpMyAdmin como usuario iwb_joomla, elegir la base de datos iwb_joomla y exportar las tablas con el método rápido al fichero iwb_joomla.sql.
