En este ejercicio hay que acabar entregando:

  • las seis capturas de pantalla
  • la copia de seguridad de los datos de la aplicación
  • la carpeta de la aplicación (comprimida en un zip)

1. Descargar la aplicación

  • Aplicación: MantisBT 1.2.14 (30/01/13). MantisBT es una aplicación de seguimiento de fallos de programas.
  • Descomprimir MantisBT en la carpeta mantisbt. La aplicación se abrirá en el navegador con la dirección mantisbt/index.php.

2. Instalar MantisBT

  • Entrar en phpMyAdmin como usuario root y crear un usuario de MySQL con nombre iwb_mantisbt_1, contraseña iwb_mantisbt_1 y base de datos iwb_mantisbt_1.
  • Abrir en el navegador la dirección mantisbt/index.php.
  • En la pantalla de opciones de instalación, utilizar como usuario y contraseña iwb_mantisbt_1 en los dos sitios en donde se pide un nombre usuario.
  • De forma predeterminada, el usuario administrador se llama administrator y su contraseña es root.

  • Crear dos usuarios que se llamen novato y enterado con los permisos que se muestran en la captura siguiente.

4. Crear incidencias

  • Entrar en MantisBT como usuario novato.
  • Cambiar el idioma a español.
  • Hacer una captura de toda la pantalla de entrada y guardarla como mantisbt_4.png:

  • Abrir dos incidencias, una para phpMyAdmin y otra para mantisBT de acuerdo con la captura siguiente.
  • Hacer una captura de toda la pantalla de incidencias y guardarla como mantisbt_5.png:

5. Resolver incidencias

  • Como usuario enterado, resolver las dos incidencias:
    • La incidencia de phpMyAdmin tiene arreglo. Asignarse la incidencia y dar la incidencia por resuelta con la siguiente explicación:

      He añadido esta línea al fichero config.inc.php de phpMyAdmin:

      $cfg['AllowUserDropDatabase'] = true;

    • La incidencia de MantisBT no tiene arreglo. Dar la incidencia por no corregible con la siguiente explicación y asignarla al administrador

      Creo que eso no tiene arreglo. Como solución, el administrador puede cambiar las contraseñas en la base de datos.

  • Como usuario novato, comprobar el estado de las incidencias.
  • Hacer una captura de toda la pantalla de incidencias y guardarla como mantisbt_6.png:

6. Copia de seguridad

  • Entrar en phpMyAdmin como usuario iwb_mantisbt_1, elegir la base de datos iwb_mantisbt_1 y exportar la tabla con el método rápido al fichero iwb_mantisbt_1.sql.
