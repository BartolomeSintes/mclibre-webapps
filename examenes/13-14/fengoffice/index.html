FengOffice 1 - Descargar FengOffice

  1. Fengoffice es un sistema de gestión de proyectos y trabajo en equipo.

    Aplicación: FengOffice 2.6 (16/05/14). Página web: Fengoffice

  2. Descomprimir FengOffice en la carpeta fengoffice.

FengOffice 2 - Instalar FengOffice

En este apartado se trata de poner en marcha FengOffice:

  1. Crear con phpMyAdmin un usuario de MySQL con nombre iwb_fengoffice_1, contraseña iwb_fengoffice_1 y base de datos iwb_fengoffice_1.
  2. Ejecutar desde Eclipse el archivo fengoffice/index.php.
  3. Rellenar la información necesario para instalar FengOffice. Por ejemplo:
    • Datos del usuario administrador:
      • nombre: admin
      • dirección de correo: admin@example.org
      • contraseña: admin
    • Nombre de la empresa: WebApps
  4. Entrar como administrador y hacer una captura del sitio:
