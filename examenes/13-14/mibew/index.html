  • las capturas de pantalla (que se llamen nombre_alumno_mibew_XXX.png donde XX sea el número de ejercicio).

  • la carpeta de la aplicación (comprimida en un zip).

1. Descargar la aplicación

  • Mibew Web Messenger es un sistema de chat para sitios web.

    Aplicación: Mibew Web Messenger 1.6.9 (24/01/14). Página web: Mibew

  • Descomprimir Mibew en la carpeta mibew. La aplicación deberá abrirse en el navegador con la dirección mibew/index.php.

2. Instalar Mibew

  • Entrar en phpMyAdmin como usuario root y crear un usuario de MySQL con nombre iwb_mibew_1, contraseña iwb_mibew_1 y base de datos iwb_mibew_1.
  • Leer las instrucciones de instalación en el archivo README de Mibew.
  • Editar el archivo de configuración con los datos del usuario de MySQL.
  • Instalar Mibew.
