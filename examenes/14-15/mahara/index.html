1. Instalar Mahara

  • Mahara es una aplicación para e-portfolios enfocada a entornos educativos.

    Aplicación: Mahara 1.10.2 (13/01/15). Página web: Mahara

  • Descomprima Mahara (la carpeta htdocs debe ser la carpeta /mahara).
  • Entre en phpMyAdmin como usuario root y cree un usuario de MySQL con nombre iaw_mahara_1, contraseña iaw_mahara_1 y base de datos iaw_mahara_1.
  • Ejecute en phpMyAdmin la orden: 
    ALTER DATABASE iaw_mahara_1 CHARACTER SET utf8 COLLATE utf8_unicode_ci;
  • Cree la carpeta /maharadata.
  • Haga una copia del fichero config-dist.php que se llame config.php. Defina en ese fichero la configuración: 
    $cfg->dbtype   = '...';
$cfg->dbhost   = 'localhost';
$cfg->dbport   = null;
$cfg->dbname   = '...';
$cfg->dbuser   = '...';
$cfg->dbpass   = '...';

$cfg->wwwroot = 'http://localhost/ ... /mahara/';

$cfg->dataroot = 'C:\ ... \maharadata';
  • Abra en el navegador /mahara y compruebe que se puede iniciar la instalación (haga una captura de toda la pantalla):

  • Instale Mahara (haga una captura de toda la pantalla confirmando la instalación):

  • Entre en Mahara y establezca adminadmin como contraseña del usuario admin (haga una captura de toda la pantalla):

  • Cambie la imagen del usuario admin a (haga una captura de toda la pantalla):
