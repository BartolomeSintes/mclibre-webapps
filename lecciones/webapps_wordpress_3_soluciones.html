En esta lección se proponen soluciones detalladas de los ejercicios (3) de WordPress. Se recomienda intentar realizarlos primero sin recurrir a estas soluciones.

WordPress (3) 1 - Widgets

  • Distribuir widgets en la parte inferior
    • Abra el menú "Apariencia > Widgets" y arrastre los widgets del cuadro "Barra lateral" a los cuadros "Pie de página 1" o "Pie de página 2".
  • Modificar widgets
    • Para eliminar un widget, haga clic en el icono del triángulo situado en su parte derecha y haga clic en el enlace Borrar.
    • Para modificar un widget, haga clic en el icono del triángulo situado en su parte derecha y modifique los campos deseados.
  • Activar widgets
    • Para activar un widget, arrastre el widget desde la lista de widgets disponibles al cuadro deseado y modifique los campos necesarios.
  • Widget "Image Widget"
    • Abra el menú "Plugins > Añadir nuevo" y escriba el nombre del widget en la caja de búsqueda situada arriba a la derecha.
    • Haga clic en el botón "Instalar ahora" del plugin "Image widget".
    • Una vez instalado, haga clic en el botón Activar.
    • Una vez activado, se mostrará la lista de plugins instalados, con el plugin "Widget de una imagen".
    • Abra el menú "Apariencia > Widgets" en el que ahora se mostrará el widget "Image Widget" en la lista de widgets disponibles.
    • Arrastre el widget al cuadro "Barra lateral".
    • Haga clic en el botón "Select an image" del widget para seleccionar una imagen de la Biblioteca multimedia o subir una nueva. Una vez subida complete los campos necesarios.
    • Haga clic en el botón Guardar del widget.
    • Comprueb
  • Widget Meta
    • Cambie el título del widget en "Apariencia > Widgets".
    • Para eliminar dos elementos del widget, debe editar el archivo con cualquier editor de texto plano (Notepad++, etc.)
    • En el archivo default-widgets.php se puede ver en qué fichero está configurado el plugin Meta 
      /** WP_Widget_Meta class */
require_once( ABSPATH . WPINC . '/widgets/class-wp-widget-meta.php' );
    • En ese archivo class-wp-widget-meta.php, el fragmento de código que genera los elementos de la lista es el siguiente: 
      <ul>
<?php wp_register(); ?>
<li><?php wp_loginout(); ?></li>
<li><a href="<?php echo esc_url( get_bloginfo( 'rss2_url' ) ); ?>"><?php _e('Entries <abbr title="Really Simple Syndication">RSS</abbr>'); ?></a></li>
<li><a href="<?php echo esc_url( get_bloginfo( 'comments_rss2_url' ) ); ?>"><?php _e('Comments <abbr title="Really Simple Syndication">RSS</abbr>'); ?></a></li>
<?php
/**
 * Filters the "Powered by WordPress" text in the Meta widget.
 *
 * @since 3.6.0
 *
 * @param string $title_text Default title text for the WordPress.org link.
 */
echo apply_filters( 'widget_meta_poweredby', sprintf( '<li><a href="%s" title="%s">%s</a></li>',
  esc_url( __( 'https://wordpress.org/' ) ),
  esc_attr__( 'Powered by WordPress, state-of-the-art semantic personal publishing platform.' ),
  _x( 'WordPress.org', 'meta widget link text' )
) );

wp_meta();
?>
</ul>
    • Elimine los dos últimos elementos 
      <ul>
<?php wp_register(); ?>
<li><?php wp_loginout(); ?></li>
<li><a href="<?php echo esc_url( get_bloginfo( 'rss2_url' ) ); ?>"><?php _e('Entries <abbr title="Really Simple Syndication">RSS</abbr>'); ?></a></li>
</ul>

      Nota: Cuando el usuario no está registrado, el código <?php wp_loginout(); ?> no genera ningún texto.
