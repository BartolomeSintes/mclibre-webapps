Cambiar la resolución de la pantalla

Referencia: AskUbuntu #128309

Inicialmente la resolución de la pantalla suele ser 640x480 px. Para ver las resoluciones posibles:

  • Inicie Ubuntu. Si no se muestra el menú de grub, inicie Ubuntu manteniendo pulsado Shift.
  • Una vez se muestre el menú de grub, pulse c y escriba los comandos 
    set pager=1    (To enable paging of long vbeinfo output)
vbeinfo
reboot         (When done)

Para aumentar la resolución:

  • Edite el fichero /etc/default/grub: 
    sudo nano /etc/default/grub
  • En ese fichero modifique la variable de configuración GRUB_CMDLINE_LINUX_DEFAULT y añada las variables GRUB_GFXMODE y GRUB_GFXPAYLOAD_LINUX 
    GRUB_GFXMODE=1024x768  # ancho x alto deseado
GRUB_CMDLINE_LINUX_DEFAULT="nomodeset"
GRUB_GFXPAYLOAD_LINUX=keep
  • Actualice grub: 
    sudo update-grub
  • Reinicie Ubuntu: 
    sudo reboot
