Bitnami (2) 2 - phpMyAdmin

  • Entrar en la máquina virtual

    El nombre del usuario administrador y su contraseña se muestran en la pantalla inicial de la máquina virtual:

  • Reconfigurar teclado

    La máquina virtual de infrestructura viene configurada de forma predeterminada con un teclado inglés. Para cambiar la configuración del teclado al teclado español, instale console-setup:

    sudo apt install dialog
sudo apt install console-setup
  • Habilitar SSH en la máquina virtual

    El servidor SSH está deshabilitado de forma predeterminada. Para habilitarlo:

    • En Debian: 
      sudo rm -f /etc/ssh/sshd_not_to_be_run
sudo systemctl enable ssh
sudo systemctl start ssh
    • En Ubuntu: 
      sudo mv /etc/init/ssh.conf.back /etc/init/ssh.conf
sudo start ssh
  • Instalar PuTTY

    La página web de PuTTY es https://www.chiark.greenend.org.uk/~sgtatham/putty/.

    En cdlibre.org hay una sección dedicada a Otras herramientas de Internet, con información detallada sobre la última versión publicada para Windows.

  • Conectar el ordenador anfitrión con la máquina virtual con Putty con túnel SSH
    • En Session ponga nombre a la sesión y guárdela:

    • En Connection > Data escriba el nombre del usuario:

    • En Connection > SSH > Tunnels añada un nuevo puerto (origen: 8888; destino: localhost:80)

    • Compruebe que se ha añadido el nuevo puerto

    • En Session guarde de nuevo la configuración y abra la sesión:

    • Al abrir la conexión, acepte la clave del servidor:

    • Si la primera vez, la máquina rechaza la conexión y se muestra un mensaje de error, vuelva a abrir Putty, cargue la conexión guardada y ábrala de nuevo:

    • Si la sesión se abre correctamente, compruebe que puede conectarse como usuario bitnami:

  • Abrir phpMyAdmin

    Abra phpMyAdmin en la máquina virtual a través de la sesión SSH escribiendo en el navegador del ordenador anfitrión la dirección http://127.0.01:8888/phpmyadmin/. El usuario administrador predeterminado de la base de datos es root con contraseña bitnami.

Bitnami (2) 3 - OpenCart

  • Descargar y ejecutar el instalador de módulo

    Para descargar e instalar un módulo, primero hay que ver la dirección del instalador del módulo en la página web de bitnami https://bitnami.com/stack/lamp/modules

    A continuación se puede utilizar el comando wget (hay que moverse antes a la carpeta en que queramos realizar la descarga):

    cd /tmp
wget https://bitnami.com/redirect/to/169794/bitnami-opencart-3.0.2.0-4-module-linux-x64-installer.run

    Y por último cambiar los permisos del archivo y ejecutarlo:

    chmod +x ./bitnami-opencart-3.0.2.0-4-module-linux-x64-installer.run
sudo ./bitnami-opencart-3.0.2.0-4-module-linux-x64-installer.run
  • Instalar la aplicación
    • Installation folder: /opt/bitnami.
    • Admin account: admin (por ejemplo)
    • La contraseña predeterminada de usuario root MySQL es bitnami
    • Hostaname: la IP de la máquina virtual
    • Mail support: n
    • Launch: Y

    La contraseña predeterminada del usuario administrador de la aplicación será bitnami

  • Apagar la máquina virtual

    Para apagar la máquina virtual, se puede utilizar el comando poweroff:

    sudo poweroff
