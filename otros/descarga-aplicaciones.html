En general, conviene utilizar siempre las últimas versiones de cualquier aplicación. Pero como los ejercicios propuestos en estos apuntes se han preparado utilizando una versión determinada de cada aplicación (normalmente, la más reciente disponible en el momento de elaboración de cada lección), es posible que los ejercicios no se puedan realizar en versiones posteriores o el interfaz de la aplicación haya cambiado y no se puedan seguir los ejercicios.

Por este motivo, en esta página se recopilan enlaces de descarga a las versiones de las aplicaciones utilizadas al elaborar estos apuntes. Estas versiones se ofrecen únicamente para la realización de los ejercicios. Para cualquier otro fin, se recomienda la descarga de las últimas versiones disponibles en la página web oficial de cada aplicación.

El problema con las versiones antiguas de las aplicaciones es que pueden ser incompatibles con los servidores actuales. Por ejemplo, algunas aplicaciones antiguas utilizaban la biblioteca mysql de PHP para acceder a MySQL, pero esta biblioteca ya no está disponible a partir de PHP 7. Si la aplicación no se puede configurar para utilizar al menos la biblioteca mysli de PHP, no será posible instalar la aplicación.
