Instaladores

  • TurnKey Linux: https://www.turnkeylinux.org/ (similar a Bitnami)

    TurnKey Linux Virtual Appliance Library nació en 2008 como proyecto de software libre dirigido a facilitar el despliegue y configuración de aplicaciones web en Debian (hasta 2012, en Ubuntu). Actualmente también ofrece instaladores para distintas plataformas de virtualización (Docker, OpenStack, etc.)
